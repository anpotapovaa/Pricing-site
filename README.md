#Pricing site

Учебный проект — сайт с тарифными планами и раскрывающимися блоками информации.  
Проект направлен на практику HTML, CSS и адаптивной верстки.  

## Технологии

- HTML  
- CSS  
- Адаптивная верстка  

## Функциональность

- отображение тарифных планов  
- раскрывающиеся блоки с дополнительной информацией  
- адаптивная верстка для разных размеров экрана  

##  О проекте

Проект выполнен самостоятельно с целью закрепления навыков HTML и CSS, а также освоения адаптивной верстки.  

## Возможные улучшения

- добавить анимации при раскрытии блоков
